# 游戏

## 如龙0 誓约的场所

上一次打开如龙0还是小半年前，打了个开头就没有继续了。至于重拾的机缘，一是看见如龙0/极1/极2即将出库，二是某一天心情低落，当时玩的血源又十分坐牢，急需寻找一个电子沙包。

于是我把只打了个开头的存档翻了出来。一开始不会点技能，在揍了30个人之后被一个小boss拿刀捅死了。重打了一遍打赢久濑之后，我想我发现这个游戏的魅力在哪里了。

个人认为如龙0的故事在整个如龙系列里应该算是最好的（前提我觉得很多代如龙的剧情都不及格）。围绕“空白的一坪”展开的双主角双线叙事。剧情节奏紧凑，悬念迭出，像看了一场酣畅淋漓的黑道电影。如龙系列或许被诟病播片过多，但恰好很适合我这类工作日非常疲惫的社畜。

我尤其偏爱如龙0里真岛的人物塑造，初次出场就能够感受到人物的性格张力，与佐川的互动也能看到隐忍之下与足够疯狂的本性的碰撞。结局时初次见到嶋野狂犬形态的真岛难免惊讶，但细想会发现脱离控制释放本性之后的他的确是这样的人。我个人很难将他对牧村实的感情简单的定性为爱，更像是在疯狂到不顾一切的反叛下，在救赎的同时完成了一场自赎。

另外，如龙的小游戏是真的很好玩。在我狂玩迪斯科和卡拉ok后，没想到真正的时间杀手是公关俱乐部小游戏，我在出库的前一个周末连玩了十个小时，终于肝完了所有的挑战。

很遗憾的是我在极2出库前只玩了一个开头，其实我能感到极2对上了我的电波，没有打完真的挺可惜，希望还会回二档库吧。另外8月我还通关了如龙6，但总体给我的感觉一般，就不写了。

# 阅读

## 《圣母》

因为这本是叙述性诡计作品，所以折叠一下相关的剧透内容。

{{< spoiler text="剧透部分折叠" >}} 

我个人其实不是很喜欢这种伦理类的叙诡，尤其反感未成年少女生子的剧情。不知道日本人究竟为什么这么热衷于未成年生子，光是8月我接触的作品就有如龙6/圣母/推子包含这样的情节。未成年少女在他们笔下完全被客体化成了天真愚蠢的胎儿容器，我光是看到“不愿意放弃还未成型的小生命”这几个字就觉得令人作呕。<br>

回到《圣母》这本书，作者努力合理化14岁少女被强奸后生子的行为——少女的母亲是不孕症患者，经历了多年治疗，流产过多个孩子之后终于得到了现在的女儿。所以觉得女儿被强奸之后怀上的小孩也是宝贵的小生命，要生下来，不可以打掉。看到这里我的怒气简直无以言表，在你的女儿怀上孩子那一刻，她在你眼里究竟是你苦苦求到的珍视的宝物，还是一个子宫，一个胎儿的容器？<br>

如果单单谈论叙述性诡计本身的话，作者的确骗到了我这种新手读者。描述凶手的时候始终不使用人称代词确实引起了我的警惕，但漫长的叙述过后我还是被糊弄过去了。揭露真相时确实一再感受到了“难道是这样？”“原来如此”的惊讶，但这种非正常伦理下模糊年龄差的欺骗方式，我实在很不喜欢。

# 观影
## 封神

我真的，我周一的时候嘲笑准备去看封神的朋友厕品，周五就被迫去看封神了。这种情况就叫做孽力回馈。

坐在影厅的两个多小时，我的感受是如芒在背，如坐针毡，如鲠在喉。这么烂的电影为什么非看不可，我为什么不在电影院门口坐两个小时的按摩椅？

如果说开场我还有质问导演的余力：谁告诉你商周对待质子的方式是让他们当兵参战，还能携带武器进入天子的宴会场所？看到后面我已经彻底失语，灾难般的烂片，立不住的人物，毫无逻辑的剧情，看完只想把导演杀了。最后看到姬发带着自动寻路马回家，连人带马从千丈悬崖落水，躺着的马啪的一下站起来继续自动寻路，我真的出离愤怒了，哪怕是塞尔达传说，马从这个高度掉下去也会死！！！

如果让我自己选，我是绝对不会去看今年暑期档的任何一部国产片的。结果还是不得已去看了这样一部片，虽然没花钱，但是遭受了极大的精神损失。这种绝世大烂片如果能继续上映第二部第三部，每一个进过电影院的观众都有责任。
